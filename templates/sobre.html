Sobre o Sistema

Este sistema foi desenvolvido com o framework Flask com o objetivo de demonstrar uma aplicação web estruturada, organizada e preparada para expansão.

Tecnologias utilizadas Python • Flask • HTML5 • CSS3

O projeto serve como base para estudos e pode ser evoluído para incluir autenticação de usuários, integração com banco de dados, relatórios gerenciais e painel administrativo.
